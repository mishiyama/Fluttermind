## Running the Project

To run  the project , open a terminal and run:

1. Clone the repo:
```bash
git clone https://github.com/mishiyama/Fluttermind.git
```
2. Navigate to the project folder:
```bash
cd Fluttermind/TicTacToeFlutter
```

3. Get dependencies:
 ```bash
flutter pub get
```
4. Run the app:
```bash
flutter run
```
